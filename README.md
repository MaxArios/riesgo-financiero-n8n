# riesgo-financiero-n8n
Proyecto de lógica de decisión para clasificar el riesgo financiero final a partir de dos variables macroeconómicas:  Riesgo Dólar  Riesgo Inflación. La lógica está pensada para ser implementada en n8n, pero es totalmente transferible a otros motores de reglas, lenguajes o flujos de automatización.
